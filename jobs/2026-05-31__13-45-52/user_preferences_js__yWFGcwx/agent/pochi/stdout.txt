# Manage User Notification Preferences using JS SDK ## Background MagicBell includes built-in user notification preferences. Users can customize which delivery channels (such as `email` or `in_app`) are enabled for different categories of notifications (such as `billing` or custom categories). These preferences are scoped per user and can be retrieved and modified securely from the client-side/user context using a User JWT and the MagicBell User SDK. In this task, you will write a Node.js program that generates a secure User JWT, initializes the MagicBell JS SDK `UserClient`, fetches the user's initial notification preferences, updates the preferences to disable the `email` channel for the `billing` category, and verifies the update. ##
