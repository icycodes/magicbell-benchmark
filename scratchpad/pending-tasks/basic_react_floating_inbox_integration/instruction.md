MagicBell provides pre-built UI components to rapidly implement notification centers without designing custom views.

You need to implement the `FloatingInbox` component within a basic React application by wrapping the app in the MagicBell `Provider`.

**Constraints:**
- Use `@magicbell/react/context-provider` and `@magicbell/react/floating-inbox`.
- Provide a dummy `apiKey` set to "YOUR_API_KEY" and `userEmail` set to "user@example.com".
- Do NOT implement HMAC security for this introductory task.
